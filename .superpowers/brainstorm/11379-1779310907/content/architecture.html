Architecture & Data Flow

Single Pages Function handles the full pipeline. API key stored as Cloudflare secret.

Browser (form)
  ├─ User pastes article URL
  ├─ Turnstile challenge
  └─ POST /api/enrich

Pages Function: /api/enrich
 ├ 1. Validate: rate limit (10/hr/IP), word cap (5K), Turnstile token
 ├ 2. Fetch article: proxy GET the URL, extract plain text
 ├ 3. Match: search pre-generated data-points.json for relevant entries
 ├ 4. Prompt: send article + matched data points to DeepSeek V4 Flash
 ├   (10K token budget, structured output format)
 └ 5. Return JSON: { original, enriched, sources[] }

Browser (render)
 └─ Side-by-side display with highlighted injections + source links

Pre-processing (one-time build step)

Source reports (.md) → LLM extraction → reports/data-points.json

JSON schema:
{
  "fact": "Only 12% of content achieves an IGS above 0.7",
  "source": "searchbloom.com/blog/information-gain-seo",
  "category": "statistic",
  "context": "Information Gain Score tiers"
}
