How should enriched results be displayed?

User pastes an article URL, tool returns the enriched version. Pick the layout that feels most useful.

A

Side-by-Side Split

Original text on the left, enriched text on the right. Injected data points are highlighted with source links. Best for direct comparison.

B

Inline With Highlights

Single text output where new data points appear inline, styled differently (e.g. amber background/callouts) with source citations. The original article is shown collapsed/expandable above.

C

Table of Changes

A structured table with columns: Original Phrase / Injected Data Point / Source Report. Best for reviewing exactly what changed and where.
